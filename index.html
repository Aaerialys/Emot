Welcome to the Aaerial Emote Collection

This website contains my collection of various emoji that I use. They can be used on discord, slack, facebook, messenger, instagram, or anywhere else that allows links to images. To use these emotes, simply type https://emot.cf/ + [emote name]. To view the full size image, use https://emot.cf/big. All emotes are of either .png or .gif file type.

The list of emotes can be found at https://github.com/Aaerialys/Emot. If you believe an emote should be added, removed, or renamed, please make a push request to the github repository.
